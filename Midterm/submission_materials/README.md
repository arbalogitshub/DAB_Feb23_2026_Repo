# 📊 Turning Dirty Data into Business Insights: A Café Sales Analysis
This data excites me because I would to change domains from being a data analyst in higher education, but rather shift my focus towards the business domain. I would like to answer business questions such as the following:

* What's the most popular item (based on sales) in the cafe?
* For the 2023, what was the best and worst months in sales?

## 📌 Overview 
This project analyzes cafe sales to uncover insights that support data-driven decision-making.

The analysis follows a full workflow from raw data to insights, including data exploration, cleaning, SQL analysis, and visualization.
